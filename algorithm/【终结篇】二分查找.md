# 【二分查找】

## 算法说明

- 首先，二分查找也称为折半查找 (Binary Search)，它是一种效率较高的查找方法，但是二分查找要求却极为苛刻，所要搜索的目标必须顺序存储，且目标序列有序

- 算法比较次数：a < log2(n + 1) < b

- 时间复杂度：O(logn)

- 空间复杂度：O(1)
